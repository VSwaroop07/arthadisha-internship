# ArthaDisha Training - C#

This repository contains practice exercises and code samples developed while learning the fundamentals of the C# programming language and the .NET ecosystem.

## Topics Covered

- **C# Basics**: Syntax, variables, data types, and operators.
- **Control Flow**: Implementation of conditional statements (`if-else`, `switch`) and loops (`for`, `while`, `foreach`).
- **Object-Oriented Programming (OOP)**: Practical application of classes, objects, inheritance, polymorphism, encapsulation, and abstraction.
- **Collections**: Working with arrays, lists, dictionaries, and LINQ.
- **Exception Handling**: Using `try-catch-finally` blocks to manage runtime errors.
- **File I/O**: Reading from and writing to files.

## Getting Started

### Prerequisites

- [.NET SDK](https://dotnet.microsoft.com/download)
- [Visual Studio Code](https://code.visualstudio.com/) or [Visual Studio IDE](https://visualstudio.microsoft.com/)

### Running the Code

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/ArthaDisha-Training-Csharp.git
   ```
2. Navigate to the project directory:
   ```bash
   cd ArthaDisha-Training-C#
   ```
3. Run the application:
   ```bash
   dotnet run
   ```
```
