# Invoice Page

A lightweight, responsive invoice generator built using pure HTML, CSS, and JavaScript. This project allows users to create, preview, and print professional invoices directly from the browser without any backend dependencies.

## Features

- **Dynamic Item Management**: Add or remove line items with real-time updates.
- **Automatic Calculations**: Instant calculation of subtotals, taxes, and grand totals.
- **Print-Ready Layout**: Optimized CSS media queries for high-quality PDF generation and physical printing.
- **Responsive Design**: Clean UI that adapts to various screen sizes.
- **Zero Dependencies**: Built with vanilla technologiesâ€”no frameworks or external libraries required.

## Getting Started

1. Clone or download the repository.
2. Open `index.html` in any modern web browser.
3. Fill in the invoice details and click "Print" or use `Ctrl+P` / `Cmd+P` to save as PDF.

## Project Structure

- `index.html`: The semantic markup for the invoice structure.
- `style.css`: Custom styling for the web interface and print-specific layouts.
- `script.js`: Logic for dynamic DOM manipulation and arithmetic calculations.
