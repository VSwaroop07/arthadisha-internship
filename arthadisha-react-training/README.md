# Arthadisha React Training

This repository contains a series of tasks and exercises designed to master React fundamentals, with a specific focus on React Hooks and modern development patterns.

## Project Overview

The goal of this project is to document the learning journey and implementation of various React features assigned during the training program.

## Topics Covered

### React Hooks
- **useState**: Managing local component state.
- **useEffect**: Handling side effects, API calls, and subscriptions.
- **useContext**: Managing global state without prop drilling.
- **useReducer**: Handling complex state logic.
- **useMemo & useCallback**: Performance optimization and memoization.
- **useRef**: Accessing DOM elements and persisting values across renders.

### Other Concepts
- Component Lifecycle
- Conditional Rendering
- List and Keys
- Form Handling
- Custom Hooks

## Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/arthadisha-react-training.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```

### Running the App
To start the development server:
```bash
npm start
```
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

## Scripts

- `npm start`: Runs the app in development mode.
- `npm test`: Launches the test runner.
- `npm run build`: Builds the app for production.
