# MSSQL Practice Queries

This repository contains a collection of Microsoft SQL Server (MSSQL) queries practiced to master database management, data manipulation, and advanced querying techniques.

## Contents

- **Basic Queries**: SELECT, WHERE, ORDER BY, DISTINCT, TOP.
- **Joins**: INNER JOIN, LEFT JOIN, RIGHT JOIN, FULL OUTER JOIN, and CROSS JOIN.
- **Aggregations**: GROUP BY, HAVING, SUM, AVG, MIN, MAX, COUNT.
- **Subqueries & CTEs**: Nested queries and Common Table Expressions.
- **Window Functions**: ROW_NUMBER, RANK, DENSE_RANK, LEAD, LAG.
- **Data Definition Language (DDL)**: CREATE, ALTER, DROP, TRUNCATE.
- **Data Manipulation Language (DML)**: INSERT, UPDATE, DELETE.
- **Stored Procedures & Functions**: Creating reusable SQL logic.
- **Indexes & Performance**: Creating indexes and analyzing execution plans.

## How to Use

1. Ensure you have **SQL Server Management Studio (SSMS)** or **Azure Data Studio** installed.
2. Connect to your MSSQL instance.
3. Open any `.sql` file from this directory.
4. Execute the queries against your practice database.

## Prerequisites

- Microsoft SQL Server (Express, Developer, or Enterprise edition).
- Sample databases like `AdventureWorks` or `Northwind` (optional, used in some scripts).
